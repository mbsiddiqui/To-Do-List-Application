To-Do List Application



Guide:

Task name must not exceed 30 characters.

Add to list - This button will add whatever task you entered in the left hand input field, into the 'To-Do' list.

Clear List - This button will remove all tasks and clear the entire List.

Done - This button will mark the respected task as done.

Edit - This button will let you edit the respected task. You'll have to press 'Add to List' button to add the updated task back to list. (Task will be added in the end of current list)

Delete - This button will let you delete the respected task from the list. (Task once deleted can't be recovered)
